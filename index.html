Guess The Word
Introduction

The project is a simple guess the word game, where the player will be provided with 8 lifes.

The user must submit only one character, otherwise error messages will be thrown underneath the letters output (no life will be deducted).

The user will be able to see, thanks to the letters button, which letters have been provided as well as the correct or wrong guess.

Expense Tracker
Introduction

The project is designed to allow the user to insert items that have been either purchased or sold.

The user needs to provide the details in the bottom section; respectively, the item name, the cost, that can be a decimal number (don't use comma but full-stop for the decimal point), and select either the expense or earning option.

The project will store the items, and the respective data, in the local-storage of the browser rendering the project itself.
Since it is the local-storage, the data will be persistent unless the user cleans it from the dev-tools or removes the items from the project.

To place multiple same items, I would suggest to add the number preceded with an underscore as the local-storage does not store muliple keys with the same name (Example: Bread, Bread_2).
